# **App Name**: TextToolPro

## Core Features:

- Fancy Text Generator: Real-time generation of 70+ Unicode-based text styles (e.g., Bold, Italic, Script, Glitch) with immediate preview and individual or bulk copy functionality, including pre-filled demo text.
- Core Text Transformation Tools: A suite of essential text utilities including a word counter (words, characters, sentences), case converter (uppercase, lowercase, title case), space remover (extra spaces, trim), text reverser (characters, words, upside down), duplicate remover (lines, words), and a slug generator.
- Grammar & Translation Assistant: Provides basic grammar corrections and translation suggestions for input text, leveraging an underlying API or mock logic tool to analyze and improve text quality.
- Lorem Ipsum Generator: Generate customizable paragraphs of placeholder 'Lorem Ipsum' text for design and content mocking.
- NATO Phonetic Translator: Convert plain text into its corresponding NATO phonetic alphabet equivalent for clear verbal communication.
- Dynamic SEO Metadata & Structure: Automatic generation of unique titles, meta descriptions, canonical tags, Open Graph data, and structured FAQ schema (JSON-LD) for every tool page to enhance search engine visibility and ranking.
- Targeted Traffic Pages: Development of SEO-optimized landing pages (e.g., 'instagram-font-generator') that feature specific examples, provide valuable content, and strategically funnel users to the main Fancy Text Generator through prominent calls-to-action.

## Style Guidelines:

- Primary interactive color: A vibrant yet professional blue-purple (#6C26E0) to signify precision and modern elegance in a light theme. It stands out clearly against the soft background for calls to action and important elements.
- Background color: An extremely light, desaturated off-white with a subtle purple tint (#FCF8FF) provides a clean, breathable canvas for text-heavy content, ensuring high readability and a contemporary feel.
- Accent color: A dynamic, slightly brighter blue (#5C6EFF) derived from an analogous hue to the primary. This color offers visual distinction and a pleasing contrast for highlights and secondary interactive elements, ensuring no 'button hunting'.
- Headlines and body text: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, offering excellent readability across all screen sizes and maintaining a consistent professional tone throughout the tool website.
- Use a consistent set of simple, functional line-art or filled icons for clarity. Icons should be clear, easily recognizable, and scale well across all devices, supporting 'thumb-friendly' button interactions without visual clutter.
- Mobile-first, responsive grid system adhering to a universal page structure including HEADER, HERO, MAIN TOOL UI, FEATURES, EXAMPLES, CONTENT, RELATED TOOLS, and FOOTER. Desktop layouts will be centered with a max-width of 1200px. All input/output controls are tightly coupled with their respective sections to ensure an intuitive 'Input → Output → Copy' flow with no large spacing gaps.
- Subtle, performant micro-animations and transitions to provide immediate feedback on user interactions (e.g., button clicks, copy success). Focus on speed and efficiency to support a fast interaction time (<5 seconds), avoiding heavy JavaScript animations.
